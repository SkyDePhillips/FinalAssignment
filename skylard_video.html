HTML <video> Tag

The HTML <video> tag is used to embed video content into a web page. It provides a way to play videos
without requiring third-party plugins like Flash. The <video> element supports multiple formats and includes
controls for playback.

Basic Syntax

<video width="600" height="400" controls>
<source src="video.mp4" type="video/mp4">
<source src="video.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>

Explanation:

  • <video> - Defines the video container.
  • <source> - Specifies multiple video file formats for browser compatibility.
  • Fallback Text - Displayed if the browser does not support the <video> tag.

Common <video> Attributes

src - Defines the video file location.

<video src="video.mp4">
controls - Adds built-in play, pause, volume, etc.

<video controls>

autoplay - Starts playing the video automatically.

<video autoplay>
loop - Replays the video continuously.

<video loop>

muted - Starts the video with no sound.

<video muted>

poster - Displays an image before the video starts.

<video poster="thumbnail.jpg">
preload - Controls how the video is loaded. Options: auto, metadata, none.

<video preload="metadata">

width & height - Sets the video dimensions.

<video width="640" height="360">

Adding Multiple Video Formats

Not all browsers support the same video formats, so providing multiple sources ensures compatibility.

<video controls>
<source src="video.mp4" type="video/mp4">
<source src="video.ogg" type="video/ogg">
<source src="video.webm" type="video/webm">
Your browser does not support the video tag.
</video>

Formats Supported by Browsers:

  • MP4 (video/mp4) - Best supported across modern browsers.
  • WebM (video/webm) - Smaller file size, good quality.
  • Ogg (video/ogg) - Open-source alternative.

Adding Captions and Subtitles

Use <track> to provide captions for accessibility.

<video controls>
<source src="video.mp4" type="video/mp4">
<track src="captions.vtt" kind="subtitles" srclang="en" label="English">
</video>

Attributes for <track>:

  • src - Link to the caption file (.vtt).
  • kind - Defines the track type (subtitles, captions).
  • srclang - Specifies the language of the subtitles.
  • label - Describes the subtitle option.

Example: Full Video Implementation

<video width="640" height="360" controls poster="preview.jpg" preload="metadata">
<source src="movie.mp4" type="video/mp4">
<source src="movie.ogg" type="video/ogg">
<track src="subtitles.vtt" kind="subtitles" srclang="en" label="English">
Your browser does not support the video tag.
</video>

Summary

<video> - Defines a video container.

<source> - Provides multiple video formats for compatibility.

<track> - Adds captions/subtitles for accessibility.
