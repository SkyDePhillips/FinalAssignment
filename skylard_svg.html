<svg> The SVG (Scalable Vector Graphics) Tag

The HTML <svg> tag is used to create vector graphics directly within an HTML document. Unlike raster
images (e.g., JPEG, PNG), SVG graphics are resolution−independent and can scale without losing quality.
They are commonly used for icons, charts, illustrations, and interactive graphics.

Basic Syntax

<svg width="200" height="100">
<rect width="200" height="100" fill="blue" />
</svg>

Explanation:

<svg> − Defines the SVG container.

<rect> − Draws a rectangle.

width & height − Set the size of the SVG and its elements.

fill − Defines the color of the shape.

Commonly Used SVG Elements

SVG graphics consist of various shape and path elements:

<circle> − Draws a Circle

<svg width="200" height="200">
<circle cx="100" cy="100" r="50" fill="red" />
</svg>
  • cx, cy − Define the circle's center position.
  • r − Specifies the radius.
  • fill − Sets the color.

<rect> − Draws a Rectangle

> <svg width="200" height="200">
<rect x="20" y="20" width="100" height="50" fill="green" />
</svg>
  • x, y − Position the rectangle.
  • width, height − Define size.

<line> − Draws a Straight Line

<svg width="200" height="200">
<line x1="10" y1="10" x2="190" y2="10" stroke="black" stroke−width="2"/>
</svg>
  • x1, y1 & x2, y2 − Define start and end points.
  • stroke − Sets line color.
  • stroke−width − Defines line thickness.

<polygon> − Draws a Polygon

<svg width="200" height="200">
<polygon points="50,10 90,90 10,90" fill="orange"/>
</svg>

points − Defines coordinates of each vertex.

<ellipse> − Draws an Ellipse

<svg width="200" height="200">
<ellipse cx="100" cy="50" rx="80" ry="30" fill="purple"/>
</svg>
  • cx, cy − Define center position.
  • rx, ry − Set horizontal and vertical radii.

<path> − Custom Paths

<svg width="200" height="200">
<path d="M10 80 C40 10, 65 10, 95 80 S150 150, 180 80" stroke="black" fill="transparent"/>
</svg>

d − Defines the path using commands like M (Move), C (Curve), L (Line), and S (Smooth curve).

Text in SVG (<text> and <tspan>)

SVG allows text rendering inside graphics.

<text> − Adding Text

<svg width="200" height="100">
<text x="10" y="50" font−size="20" fill="black">Hello SVG</text>
</svg>

x, y − Position the text. font−size − Sets the text size.

<tspan> − Multi−line or Styled Text

<svg width="200" height="100">
<text x="10" y="50" font−size="20">
<tspan fill="red">Red</tspan> and
<tspan fill="blue">Blue</tspan> Text
</text>
</svg>

<tspan> allows styling specific text portions.

Styling and Animation

SVG elements can be styled with CSS or animated.

CSS Styling

<style>
.circle { fill: yellow; stroke: black; stroke−width: 3; }
</style> <svg width="200" height="200">
<circle cx="100" cy="100" r="50" class="circle"/>
</svg>

Animation with <animate>

code> <svg width="200" height="200">
<circle cx="50" cy="50" r="20" fill="blue">
<animate attributeName="cx" from="50" to="150" dur="2s" repeatCount="indefinite"/>
</circle>
</svg>

attributeName − Defines the attribute to animate. from, to − Sets start and end values. dur − Duration of animation. repeatCount="indefinite" −
Loops animation.

Embedding External SVG Files

SVG files can be embedded using <img> or <iframe>.

Using <img>

<img src="graphic.svg" width="200" height="200" alt="SVG Graphic">

Using <iframe>

<iframe src="graphic.svg" width="200" height="200"></iframe>

Summary

<svg> − Defines the SVG container.

<circle> − Draws a circle.

<rect> − Creates a rectangle.

<line> − Draws a line.

<polygon> − Defines a polygon shape.

<ellipse> − Creates an ellipse.

<path> − Draws custom shapes using path data.

<text> − Adds text inside SVG.

<tspan> − Styles text within <text>.

<animate> − Adds animation to SVG elements.
